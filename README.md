# Credit Analysis Portfolio

Independent credit analyses conducted using US GAAP financial statements and SEC filings, applying DFAST-style stress testing methodology to evaluate corporate and sovereign credit risk.

---

## [Southwest Airlines Co. (NYSE: LUV) — Independent Credit Analysis](./Southwest-Airlines-Credit-Analysis)
6-ratio credit analysis with a 3-scenario stress test through 2028. Debt/EBITDA improved from 4.57x to 2.30x. Recommendation: Approve with Conditions.

## [Expected Loss Modeling — Multi-Tier Loan Portfolio](./Expected-Loss-Modeling)
Python model calculating expected loss across an 18-loan portfolio (BBB–CCC) using Moody's/S&P default benchmarks. *(In Progress)*

## [Sovereign CDS Data Extraction & Macro Dataset Construction](./Sovereign-CDS-Data-Extraction)
20+ years of daily CDS spreads across 19 sovereign nations (Bloomberg) plus cross-country short-term interest rates (FRED), structured into a usable panel dataset.

---

## Tools Used
Excel (Advanced — 3-tab financial models, scenario switches, sensitivity tables), Python, Bloomberg Terminal, FRED API, SEC EDGAR, DFAST-style methodology.
